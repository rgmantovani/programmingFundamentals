
Ex02: Faça um programa que leia uma string e conte a
quantidade de vogais.

[Entrada] -> processamento -> [Saidas]

Entrada: ler (obter informação - ler - scanf, gets)
 * uma string (usuário)

Processamento ...
[Entrada] -> [Saida]

  1. ler a palavra (vetor de caracteres)

  "Big Luiz" -> contador de vogais = 3
   xSxxxSSxx

  "Chinelo"  ->  contador de vogais = 3
   xxSxSxSx

   vogais = {a, e, i, o, u, A, E, I, O, U}

   2. percorrer a palavra (vetor - for)
      posição do vetor - caracter
      a) analisar se é ou não vogal
      b) se for vogal
        contar + 1
      c) se for consoante (não faz nada)

Saída: numero de vogais da palavra que lemos
